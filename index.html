Who Am I?

Hi I'm Daffa Saya adalah siswa Teknik Komputer dan Jaringan dari SMKN 13 Bandung yang memiliki antusiasme tinggi dalam administrasi sistem dan infrastruktur jaringan. Saya memiliki pengalaman praktik dalam membangun jaringan dari sisi fisik seperti instalasi Kabel UTP dan Fiber Optic, hingga konfigurasi perangkat menengah seperti Mikrotik dan penyelesaian berbagai topologi jaringan.

Selain itu, saya aktif mendalami administrasi server baik di lingkungan Windows Server (AD DS, DNS) maupun Linux Debian, mulai dari perintah dasar hingga layanan kompleks seperti Remote Server, File Server (Samba), FTP, DHCP, serta pengelolaan Web Server berbasis XAMPP dan database PHPMyAdmin.
